## 2025-05-15 - Accessible Tab Pattern
**Learning:** Converting non-semantic `div` tabs to `button` elements with the full ARIA tabbed interface pattern (role="tablist", role="tab", role="tabpanel") significantly improves screen reader and keyboard accessibility with minimal CSS overhead.
**Action:** Always use the `button` element for tabs and implement Arrow/Home/End keyboard navigation to follow standard W3C accessibility patterns.
