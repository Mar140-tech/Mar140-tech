## 2025-12-04 - [Accessible Tabs Implementation]
**Learning:** For accessible tabs, use semantic `button` elements with `role="tab"` inside a `role="tablist"`. Synchronize `aria-selected` and `active` classes in JS. Support Arrow keys, Home, and End for keyboard navigation. Use `focus-visible` for clear focus indicators.
**Action:** Apply this pattern to all tab-based navigation to ensure screen reader and keyboard accessibility.
