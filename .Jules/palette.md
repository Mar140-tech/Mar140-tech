## 2025-05-14 - Accessible Tab Navigation
**Learning:** Using `div` elements for tabs makes them inaccessible to keyboard and screen reader users. Implementing the WAI-ARIA tabs pattern with `button` elements, `role="tablist"`, and proper ARIA attributes significantly improves accessibility.
**Action:** Always use semantic `button` elements for tabs and implement keyboard navigation (arrow keys, Home/End) following WAI-ARIA guidelines.
