## 2025-05-10 - [Accessible Emojis in Markdown]
**Learning:** Emojis in Markdown are often skipped or misread by screen readers. Wrapping them in `<span>` with `role="img"` and `aria-label` is a high-impact, low-line-count micro-UX improvement for documentation.
**Action:** Use accessible spans for all decorative or meaningful emojis in READMEs and other UI text.

## 2025-05-10 - [Project Branch Visibility]
**Learning:** Users often miss work-in-progress features if they are on non-default branches. A clear "micro-UX" win is adding a direct link in the primary README to the relevant feature branch.
**Action:** Always check for relevant feature branches and provide clear pointers in the main documentation.
